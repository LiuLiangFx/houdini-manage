    Installation

    We provide prebuilt distributables on the GitHub Releases page for Windows. On other platforms, you currently need to install Houdini-Manage via the command-line:

    $ pip git+https://github.com/NiklasRosenstein/houdini-manage.git
$ houdini-manage --gui

    macOS: If you did not install Python via Homebrew, you need to add use sudo .
    Windows: Depending on where you chose to install Python, you might need an Administrator console on Windows.
